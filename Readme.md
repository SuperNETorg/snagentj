How to build

Linux:

- ./compile_native.sh
- ./compile.sh

Windows:

- Add path to Visual Studio cl.exe eg D:\Microsoft Visual Studio 11.0\VC\bin\ to system PATH variable
- Run compile_native_win32.bat
